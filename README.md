# lttf-diff

> Warning: work in progress

This action is designed to integrate [lint-to-the-future](https://github.com/mansona/lint-to-the-future) to pull request workflows:

- Display changes compared to current main branch
- Block merges that add new ignores

> **Note:** `lttf-diff` needs current Lint-to-the-Future data for the main branch (as generated by `lttf output`). By default it tries to load data from gh-pages, as published by [lttf-dashboard](https://github.com/mansona/lttf-dashboard).

```yaml
name: Lint to the Future diff check

on:
  pull_request:
    types:
      - opened
      - synchronize
      - reopened

jobs:
  lttf-diff:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: njoyard/lttf-diff@v1
```

## Parameters

| Name     | Description                                                                                                                  |
| -------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `data`   | Path or URL to fetch current main branch `data.json` file. When omitted, `lttf-diff` will try to fetch it from github pages. |
| `folder` | Github pages subfolder where the LTTF dashboard was deployed.                                                                |
| `fail`   | Set to `false` to disable failing when new ignores are detected                                                              |
